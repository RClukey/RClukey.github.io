Vending Machine

This project was designed to give the team experience in embedded systems and the development of mechatronic systems with mechanical, electrical, and computer science components. The project showed the team how these different fields of engineering came together to make one fully-functioning product.

The CAD design for the vending machine mechatronic system.


The CAD model was designed to give the fabrication team the dimensions for the physical design, and to give the electrical team an idea on how the electronics for the vending machine should be oriented for maximized space optimization.


The final design for the vending machine mechatronic system.


The fabrication of the vending machine gave the team experience in working in a machine shop, using tools like a jigsaw, a router, a lathe, and a bandsaw. This allowed the team to understand the necessity for designing the product with manufacturing in mind, so the final product isn't difficult to manufacture, driving up the cost of the overall project.


The electrical schematic for the vending machine mechatronic system.


The buttons on the schematic allow the user to indicate which item they wish to purchase. purchase The interaction with the buttons tells the Arduino to proceed to the second step of the finite state machine. The IR sensor detects the input of coins for payment. Once the correct amount of payment has been processed, the Arduino proceeds to the third stage of the finite state machine. The PCA9685 servo motor driver powers the corresponding servo motor for to dispense the item the customer bought. Before returning the finite state machine to the first stage, waiting for a user to input an item. With the LEDs indicating which state the vending machine is in.
