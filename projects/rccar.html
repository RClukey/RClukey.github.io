Remotely Operated Car

The initial electronics testing of the RC car control system.


Using an Arduino MEGA 2560, I coded the system to take as input the position of the
joystick to spin the motor for the drivetrain and rotate the servo to steer the car.
The servo motor was later replaced with a stepper motor to finely control the angle
of the front wheels. The car is designed for rear-wheel drive with each wheel receiving
the same power, regardless of the orientation of the front-wheel steering. This causes
some slipping in the model. This was later fixed in the final version.



The CAD model for the remotely operated car.


I recently designed and fabricated a fully custom remote-controlled car,
integrating mechanical design, electronics, and embedded programming.
I developed the control system using an Arduino microcontroller and created
the mechanical structure in CAD. SolidWorks allowed me to model the assembly
and run simulations to evaluate performance under various loading conditions.



To bring the design to life, I sourced components through 3D printing, McMaster-Carr,
and SendCutSend, ensuring each part met the functional requirements of the system.
Using the Arduino IDE, I programmed the vehicle in C++ to operate via joystick
input and added a cruise control feature that maintained a constant speed and
direction without manual adjustment. This project strengthened my skills in
mechanical design, embedded systems, and power management, and it now serves as
the foundation for my next goal: developing my own electric skateboard.
